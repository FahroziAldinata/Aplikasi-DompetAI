# 📱 Panduan Install APK DompetAI

## Download
| Versi | Link | Ukuran |
|---|---|---|
| Release v1.0.0 | [app-release.apk](../build/app/outputs/flutter-apk/app-release.apk) | ~15MB |

## Cara Install
1. Download file APK
2. Buka **Pengaturan → Keamanan**
3. Aktifkan **"Izinkan sumber tidak dikenal"**
4. Buka file APK yang didownload
5. Tap **Install**

## Persyaratan
- Android 6.0 (API 23) ke atas
- RAM minimal 2GB
- Storage 50MB

## Izin Aplikasi
| Izin | Kegunaan |
|---|---|
| Storage | Simpan data transaksi lokal |
| Notifications | Pengingat jatuh tempo utang |

## Privasi
Semua data tersimpan **lokal di HP kamu**.
Tidak ada data yang dikirim ke server manapun.
Model AI berjalan 100% offline.

## Troubleshoot
**App tidak bisa diinstall?**
→ Pastikan izin sumber tidak dikenal aktif

**App crash saat buka?**
→ Paksa berhenti → hapus data app → buka lagi

**Parser salah baca transaksi?**
→ Coba format: "[item] [nominal] [akun]"
→ Contoh: "kopi 20rb cash"
